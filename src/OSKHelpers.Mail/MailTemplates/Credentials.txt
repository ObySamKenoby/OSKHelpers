# Credentials.txt
# Mail inviata al momento della creazione di un nuovo utente o all'aggiornamento forzato della password (pannello amministratore o recupero).

# Tags disponibili:
# {ENDPOINT}   Indirizzo per poter effettuare l'accesso
# {DATE}       Data attuale
# {TIME}       Ora attuale
# {FULLNAME}   Nome completo dell'utente
# {AZIENDA}    Codice dell'azienda 
# {RAGSOC}     Ragione sociale
# {USERNAME}   Nome utente
# {PASSWORD}   Password

# Condizioni disponibili
# <<ISNEW>>    Il contenuto viene mostrato quando l'utente è nuovo (è stato appena inserito)
# <<ISNOTNEW>> Il contenuto viene mostrato quando l'utente non è nuovo (è stato aggiornato)
# <<NOTSINGLEAZIENDA>> Il contenuto viene mostrato quando l'applicativo non è configurato per gestire una singola azienda

@Subject: {RAGSOC} - Invio credenziali accesso per {FULLNAME}

Spett.le {FULLNAME}<br/>
<br/>
con la presente le comunichiamo che sono state abilitate per permetterle l'accesso alla piattaforma G.F.V.<br/>
<br/>
I dati per poter accedere sono i seguenti:<br/>
Pagina di login: <a href="{ENDPOINT}">{ENDPOINT}</a><br/>
<<NOTSINGLEAZIENDA>>Azienda: <strong>{AZIENDA}</strong><</NOTSINGLEAZIENDA>><br/>
Username: <strong>{USERNAME}</strong><br/>
Password: <strong>{PASSWORD}</strong><br/>
<br/>
E' consigliata la modifica della password al primo accesso.<br/>
Le auguriamo buona giornata e buon lavoro.
